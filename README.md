# 马振奥的学赚助手

一站式学业管理 + 接单工具 + 自动运营平台

## 功能

- 📊 **工作台** - 数据概览，快速添加任务/笔记/记账
- 📚 **学业管理** - 待办任务、学习笔记、闪卡记忆、课程表
- 💼 **接单工具** - 网页采集、Excel处理、PDF生成、报价参考
- 🤖 **运营中心** - 自动发帖、客户跟进
- 💬 **AI 聊天** - 智能对话助手
- 📈 **赚钱情报** - 搜集和发布赚钱信息
- 💰 **财务管理** - 收支记录
